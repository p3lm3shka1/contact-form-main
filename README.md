## Built with

- **HTML5** — page markup and form structure
- **SCSS (Sass)** — styling (compiled to CSS)
- **Vanilla JavaScript** — validation + UI interactions (errors / toast)
- **npm** — scripts for compiling SCSS

---

## Features

- Required field validation
- Email format validation 
- Query type validation
- Consent validation
- Error styling via the `has-error` CSS class
- Success toast (“Message Sent!”) on valid submission

---

## Project structure

- `index.html` — main page with the form
- `js/`
  - `form-submit.js` — validation logic and toast behavior
- `scss/` — SCSS source files
- `css/` — compiled CSS output (generated)
- `assets/` — images/icons
- `design/` — design assets (you can also place the screenshot here)
- `package.json` — dependencies and scripts
